Câu 1. Position: relative và position: absolute

position: relative giữ phần tử ở vị trí ban đầu nhưng có thể dịch chuyển bằng top, left, right, bottom. Phần tử vẫn chiếm không gian trong bố cục.

position: absolute đưa phần tử ra khỏi luồng tài liệu và định vị theo phần tử cha gần nhất có position khác static.

Ví dụ: 
.parent{
    position:relative;
}

.child{
    position:absolute;
    top:10px;
    right:10px;
}

Ứng dụng: Relative dùng để làm phần tử cha, Absolute dùng cho icon, badge, nút nổi hoặc thông báo.

Khi deploy: Không nên lạm dụng absolute vì dễ bị lệch trên nhiều kích thước màn hình. Nên kết hợp Flexbox hoặc Grid để website responsive.

Câu 2. Domain - Hosting - DNS

Quy trình:

  1. Đăng ký tên miền (Domain).
  2. Mua hoặc sử dụng Hosting.
  3. Upload mã nguồn website.
  4. Cấu hình DNS trỏ Domain về Hosting.
  5. Kiểm tra website hoạt động.

DNS có nhiệm vụ chuyển tên miền thành địa chỉ IP của máy chủ.

Công cụ deploy miễn phí:

  • GitHub Pages
  • Netlify

Rủi ro bảo mật:

  • Cấu hình DNS sai làm website không truy cập được.
  • Không bật HTTPS khiến dữ liệu dễ bị đánh cắp.
  • Phân quyền sai có thể làm lộ mã nguồn.

Câu 3. So sánh let, const và var

Kiểu Scope Hoisting
var Function Scope Được hoisting và có giá trị undefined
let Block Scope Hoisting nhưng không được dùng trước khi khai báo
const Block Scope Không được gán lại giá trị

Ví dụ lỗi:

console.log(a);

let a = 10;

Kết quả:

ReferenceError

Debug: Mở Chrome DevTools (F12) → Console để xem lỗi hoặc dùng Breakpoint trong tab Sources để kiểm tra giá trị biến.

Câu 4. Event Delegation

Event Delegation là kỹ thuật gắn một Event Listener vào phần tử cha để xử lý sự kiện của các phần tử con thông qua cơ chế nổi bọt (Event Bubbling).

Ví dụ:

document.getElementById("list")
.addEventListener("click",function(e){

if(e.target.tagName==="LI"){
alert(e.target.innerText);
}

});

Ứng dụng:

  • Danh sách sản phẩm.
  • Danh sách công việc (Todo List).
  • Bảng dữ liệu động.

Lợi ích:

  • Ít Event Listener hơn.
  • Tăng hiệu suất.
  • Hoạt động với phần tử được thêm động bằng JavaScript.
